P-03

Statistical Analysis with NumPy

Objective: Utilize the NumPy library for generating a large dataset of random numbers and performing fundamental statistical analysis — mean, median, variance, standard deviation, minimum, and maximum values. 
import numpy as np

random_1 = np.random.random(1000)
print(random_1)

mean = np.mean(random_1)
print(mean)

median = np.median(random_1)
print(median)

var = np.var(random_1)
print(var)

min = np.min(random_1)
print(min)

std = np.std(random_1)
print(std)
P-04

Data Aggregation and Grouping using Pandas

Objective: Organize data based on specific categories and apply multiple statistical functions to analyze sales and unit distributions using Pandas. 
import pandas as pd

data = pd.DataFrame({
    'city':    ['mumbai', 'pune', 'panjab', 'kolapu'],
    'product': ['mmilk', 'apple', 'orange', 'baanana'],
    'sales':   [234, 543, 565, 765],
    'units':   [200, 400, 600, 560]
})
print(data)

f_n = ['count', 'mean', 'sum', 'std']

f_linst = data.groupby(['city', 'product']).aggregate({
    "sales": f_n,
    "units": f_n
})
print(f_linst)
P-06

Matrix Operations and Linear Algebra using NumPy

Objective: Perform fundamental linear algebra operations — matrix arithmetic, determinant calculation, and finding the multiplicative inverse — using the NumPy library. 
import numpy as np

random_1 = np.random.randint(1, 10, size=(3, 3))
random_2 = np.random.randint(1, 10, size=(3, 3))

print(random_1)
print("\n")
print(random_2)
print("\n")
print(random_1 + random_2)
print("\n")
print(random_1 - random_2)
print("\n")
print(random_1 * random_2)
print("\n")
print(random_1 / random_2)

det_a = np.linalg.det(random_1)
inv_b = np.linalg.det(random_2)
print(det_a)
print('\n')
print(inv_b)

try:
    inv_a = np.linalg.inv(random_1)
    inv_b = np.linalg.inv(random_2)
    print(inv_a)
    print('\n')
    print(inv_b)
except np.linalg.LinAlgError as e:
    print(e)
P-09

Conditional Filtering on NumPy Arrays

Objective: Perform conditional filtering on a NumPy array and calculate statistical measures (mean) on the filtered subset of data. 
import numpy as np

data_num = np.random.randint(1, 1000, size=100)

filter_value = (data_num > 500) & (data_num < 800)
print(filter_value)

filter_num = data_num[filter_value].mean()
print(filter_num)
P-10

Multi-Featured Dataset with Pandas Pivot Table

Objective: Create and display a multi-featured dataset using the Pandas library, focusing on structured data organization for business metrics using pivot tables. 
import pandas as pd

data_fr = pd.DataFrame({
    'date':    ['2019-04-05', '2023-11-20', '2026-08-12'],
    'region':  ['west', 'north', 'south'],
    'product': ['Ak-47', 'handgranet', 'lonchar'],
    'sales':   [200, 444, 666]
})

final_fr = data_fr.pivot_table(
    index='sales',
    columns='region',
    values='product',
    aggfunc='min'
)
print(final_fr)
